# Frequently Asked Questions

## General Questions

<Accordion title="What is Kart Companion?">
Kart Companion is a browser extension for Smash Karts that provides real-time statistics tracking, match history, performance analytics, and HUD overlays to enhance your gaming experience.
</Accordion>

<Accordion title="Is Kart Companion free?">
Yes, Kart Companion is completely free to download and use from the Chrome Web Store.
</Accordion>

<Accordion title="Which browsers are supported?">
Currently, Kart Companion is available for Google Chrome. Support for other Chromium-based browsers may be added in the future.
</Accordion>

## Installation & Setup

<Accordion title="How do I install Kart Companion?">
1. Visit the [Chrome Web Store](https://chromewebstore.google.com/detail/bohhelmgldjclickdnlodllfkmgfjonf?utm_source=item-share-cb)
2. Click "Add to Chrome"
3. Confirm the installation
4. The extension will automatically activate when you visit smashkarts.io
</Accordion>

<Accordion title="Why isn't the extension working after installation?">
Make sure you're on the official Smash Karts website (smashkarts.io). The extension only works on this domain for security reasons. If it's still not working, try refreshing the page or restarting your browser.
</Accordion>

## Features & Functionality

<Accordion title="What statistics does Kart Companion track?">
Kart Companion tracks:
- Kill/Death ratios
- Match history with detailed statistics
- Performance analytics
- Streak monitoring
- HUD overlays with real-time data
</Accordion>

<Accordion title="How do I view my match history?">
Your match history is automatically tracked and can be viewed by clicking the Kart Companion icon in your browser toolbar. All match data is stored locally in your browser.
</Accordion>

<Accordion title="Can I export my data?">
Yes! Kart Companion includes import/export functionality. You can export your match history and statistics as JSON files for backup or analysis.
</Accordion>

## Troubleshooting

<Accordion title="The HUD isn't showing up in-game">
1. Make sure the extension is enabled
2. Check that you're on smashkarts.io
3. Try refreshing the page
4. Ensure your browser isn't blocking popups or scripts
</Accordion>

<Accordion title="My statistics aren't updating">
Statistics update automatically during matches. If they're not updating:
1. Refresh the page
2. Check your internet connection
3. Make sure the extension has permission to access smashkarts.io
</Accordion>

<Accordion title="The extension is causing lag or performance issues">
Kart Companion is optimized for performance, but if you're experiencing issues:
1. Try the performance optimization guide in our developer documentation
2. Disable other browser extensions temporarily
3. Update to the latest version of Chrome
</Accordion>

## Privacy & Security

<Accordion title="Is my data safe?">
Yes! All data is stored locally in your browser. Kart Companion does not collect or transmit any personal information. Your match history and statistics remain private on your device.
</Accordion>

<Accordion title="Does Kart Companion work with VPNs?">
Yes, Kart Companion works with VPNs. However, some VPN configurations might affect the extension's ability to track statistics if they interfere with the game's WebSocket connections.
</Accordion>
