# Quickstart

## Installation

### Step 1: Download from Chrome Web Store
1. Visit the [Chrome Web Store](https://chromewebstore.google.com/detail/bohhelmgldjclickdnlodllfkmgfjonf?utm_source=item-share-cb)
2. Click "Add to Chrome"
3. Confirm the installation

### Step 2: Enable the Extension
1. Click the Kart Companion icon in your browser toolbar
2. The extension popup will open
3. You're ready to start tracking!

## Getting Started

### First Time Setup
When you first open the extension, you'll see:
- Welcome screen with basic instructions
- Option to import existing stats (if you have them)
- HUD customization modal

### Basic Usage
1. **Enable Tracking**: Make sure tracking is enabled in the popup
2. **Play Smash Karts**: Navigate to smashkarts.io and start playing
3. **View Stats**: Click the extension icon anytime to see your current stats
